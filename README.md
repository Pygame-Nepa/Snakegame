##  Snake Game

A snakegame which is like kepad phone style you can play here.Eat Fruits To Make Snake Bigger.

**Live Demo**: (https://snakebyarpan.netlify.app/)

## Features:
**Keyboard Control**
**Responsive UI**
**Feed Fruit To Make snake Bigger**
**Snake Will Die After Touching Border**

## Built With:

**HTML5**

**CSS3**

**JavaScript**
