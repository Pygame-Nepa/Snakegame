#  Snake Game

A sleek, responsive, and retro-inspired Snake Game built entirely using web technologies. No installations or extensions required—just open your browser and play!

Live Demo: [Play the game here](https://snakebyarpan.netlify.app/)



# Features

Classic Gameplay: Smooth mechanics and responsive controls reminiscent of the original arcade game.
Score Tracking: Real-time score updates as you collect food.
Clean UI: Simple, distraction-free retro aesthetic designed with modern CSS.
Fully Responsive: Play directly in your desktop browser.



# Built With

HTML5: Structured layout and Canvas rendering for the game grid.
CSS3: Retro styling, custom fonts, and centering layouts.
JavaScript : Game loop logic, snake movement, grid collision detection, and event listeners.
